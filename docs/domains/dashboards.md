# Dashboards

## Access Tiers
- `public`: no operational dashboards
- `network`: operational trend and monitoring dashboards
- `coordinator`: review, executive, export, and portfolio workflows

## Dashboard Inputs
- Aggregated regional activity
- time-bucket trends
- watch areas
- access entitlements
- role/profile context
- source mix and report confidence state

## Boundary
- Dashboards must be driven by aggregated data, not raw precise points.
- Dashboard labels, empty states, and exports are native in English and Spanish.
- Public dashboards are not a first-class surface; the paid value sits in organizational operations.
