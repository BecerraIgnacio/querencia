# Analytics Contracts

## Core Outputs
- aggregated regional activity
- time-bucket trend series
- KPI snapshots
- dashboard summaries
- critical-region rankings

## Rules
- Advanced metrics come from aggregated or policy-approved data views.
- Analytics payloads may include localized labels, but metric keys and dimensions remain language-neutral.
- Cross-tier analytics use the same shared contract vocabulary even when feature depth differs.
- Source mix and report confidence state are first-class operational analytics dimensions.
