# Plugin Policy

## Fixed
- `claude-mem`

## Module Persistent Defaults
- web milestone: `javascript-typescript`
- pipeline milestone: `python-development`, `data-engineering`

## Common Temporary Plugins
- frontend: `frontend-mobile-development`, `ui-design`, `unit-testing`, `tdd-workflows`
- backend: `backend-development`, `database-design`, `backend-api-security`, `security-scanning`
- analytics: `business-analytics`, `performance-testing-review`
- docs/governance: `documentation-generation`, `c4-architecture`, `comprehensive-review`

## Not Default
- `everything-claude-code` is reference-only.
- `payment-processing`, SEO plugins, and other growth plugins stay deferred until their domain is active.
